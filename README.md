## RMarkdown Workshop

This workshop is part of the USU Ecology Center Data Science workshop series. The materials are typically emailed to workshop participants before the workshop. Participants follow along in the RMarkdown file and complete the exercises to practice using RMarkdown.

The repository includes the RMarkdown file and the HTML and PDF output files. This workshop is intended for folks who are comfortable with coding and RStudio but haven't used RMarkdown before.

## Topics

* Document output and knittng
* Basic syntax 
* Adding code and math chunks
* Embedding images
